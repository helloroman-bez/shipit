# SHIP IT — Game Design Document & Task для Claude Code

## Цель
Превратить прототип геймифицированного трекера контента «Ship It» (React-компонент) в полноценное мобильное веб-приложение и задеплоить его.

## Исходник
Прототип: `ship-it.jsx` (приложен в проекте)

---

## Философия продукта

Ship It — бесконечная игра для создателей контента. Четыре барьера мешают регулярно публиковать: прокрастинация, бросание через неделю, паралич «что писать», нерегулярность. Каждый барьер закрывается отдельным слоем механик.

Принцип: приложение должно ощущаться как игра, в которую хочется зайти, а не как трекер, в который надо заставлять себя заходить.

---

## Архитектура механик — 4 слоя

### Слой 1 — Daily Engine (решает «не могу начать» и «что писать»)

Три механики работают каждый день:

- **Стрик (цепочка дней)** — счётчик непрерывных дней с публикацией. Визуально заметный, на главном экране. Работает на loss aversion: терять 23-дневный стрик больнее, чем лень писать пост.

- **Рулетка форматов** — рандомайзер задания из 20+ вариантов. Убирает паралич выбора «что писать». Анимация кручения, тактильный отклик. Каждое задание включает связку: тема + формат + платформа + готовый скелет поста (хук → проблема → решение → результат).

- **Daily quest** — каждый день генерируется 1 конкретное мини-задание. Бонус +15 XP за выполнение. Примеры: «Опубликуй пост с цифрами», «Сними 30-секундный Reels», «Напиши пост-провокацию». Обновляется в полночь.

- **Push-уведомление** — вечернее напоминание (через Notification API, с разрешением пользователя). Текст: «Стрик {N} дней — не сломай!» или «Daily quest ждёт — +15 XP». Последний триггер перед дедлайном.

- **Утренний/вечерний ритуал** — утренний экран: мотивационная фраза дня + текущий daily quest + кнопка «Начать день». Вечерний экран: итоги дня (XP за день, стрик, выполнен ли квест). Превращает приложение в привычку с чётким входом и выходом.

### Слой 2 — Progression (решает «бросаю через неделю»)

XP растёт нелинейно, награждая постоянство:

- **XP за контент** — разные форматы = разный XP:
  - Текстовый пост — 10 XP
  - Сторис-серия — 15 XP
  - Голосовое/подкаст — 20 XP
  - Видео/Рилс — 25 XP
  - Скринкаст/туториал — 35 XP
  - Кейс с цифрами — 40 XP
  - Коллаб/эфир — 50 XP
  - Мем/юмор — 10 XP

- **Streak bonus** — бонус к XP за серию дней:
  - 3+ дней подряд → +25% к XP
  - 7+ дней подряд → +50% к XP
  - 14+ дней подряд → +75% к XP
  - 30+ дней подряд → +100% к XP (x2)

- **Комбо-система** — бонус за несколько постов в один день:
  - 2-й пост за день → x1.5 XP
  - 3-й пост за день → x2.0 XP
  - 4+ пост → x2.5 XP

- **Decay штраф** — пропустил день = -10% от накопленного XP. Делает каждый пропуск болезненным — ты буквально теряешь прогресс. Отображается красным на экране: «-{N} XP потеряно».

- **7 уровней прокачки**:
  1. 🌱 Новичок (0 XP)
  2. ⚔️ Контент-боец (50 XP)
  3. 🎯 Охотник за лидами (150 XP)
  4. 🌀 Мастер воронок (350 XP)
  5. 🏗️ Архитектор систем (600 XP)
  6. 🔮 Мастер спиралей (1000 XP)
  7. 👑 Легенда Ship It (1500 XP)

### Слой 3 — Collection (долгосрочный крючок)

- **15+ достижений** — разблокируются автоматически по условиям:
  - 🚀 Первый шаг — опубликуй первый пост
  - 🔥 Неделя огня — 7 дней подряд
  - 💎 Несгибаемый — 30 дней подряд
  - 💯 Сотня — набери 100 XP
  - ⚡ Полтысячи — набери 500 XP
  - 📊 Кейсоман — 5 кейсов с цифрами
  - 🎬 Видеомейкер — 10 видео/рилсов
  - 🌐 Мультиплатформа — публикуй на 3+ площадках
  - 🏆 Полсотни — 50 постов всего
  - 🎰 Рулеточник — выполни 10 заданий рулетки
  - 🌙 Ночной шиппер — залогай пост после 23:00
  - 📱 Формат-хамелеон — все 8 типов контента за месяц
  - 🤝 Boss slayer — 3 коллаба/эфира
  - ⭐ Daily warrior — 7 daily quest подряд
  - 🎯 Снайпер — 5 постов с охватом 500+

- **Тепловая карта** — визуализация активности за 90 дней по типу GitHub contributions. Зелёные квадраты разной интенсивности. Хочется «закрасить» весь календарь.

- **Титулы** — каждый уровень имеет название и иконку. Генерируются shareable-карточки для сторис (canvas → PNG). Это и контент сам по себе, и социальное обязательство.

- **Сезоны** — 30-дневные циклы. В начале каждого сезона: новый набор целей, сезонные достижения, обнуление сезонного рейтинга (общий XP сохраняется). Поддерживает свежесть и даёт точки перезапуска.

### Слой 4 — Challenges (азарт и вызовы)

- **Недельный челлендж** — 7 из 7 дней с публикацией. Бонус: +100 XP.
- **Формат-микс** — опубликуй 5 разных типов контента за неделю. Заставляет выходить из зоны комфорта. Бонус: +150 XP.
- **Марафон** — 30 дней без пропусков. Бонус: +500 XP + специальный титул.
- **Boss fight** — коллаб или прямой эфир. Самый сложный формат, максимальный XP. Бонус: +200 XP сверх обычного.
- **Speed run** — 3 поста за 1 день. Бонус: +75 XP.
- **Челленджи отображаются на отдельной вкладке** с прогресс-барами и таймерами.

## Дополнительные системы

### Контентный помощник (Smart Roulette)
Рулетка выдаёт не просто «тип контента», а готовую связку:
- **Тема** (из пула 30+ тем по нише)
- **Формат** (текст, видео, сторис, карусель...)
- **Платформа** (Instagram, YouTube Shorts, Telegram...)
- **Скелет поста**: хук → проблема → решение → результат
- Примеры выпадений:
  - «Кейс + Reels + до/после клиента → Хук: число → Было: проблема → Сделали: процесс → Стало: результат»
  - «Мем + Instagram + боль целевой аудитории → Верх: ожидание → Низ: реальность → Подпись: CTA»

### Аналитика и фидбэк
- При логировании поста — опциональное поле «Охваты / реакции» (число)
- Экран аналитики: какие форматы дают больше охватов, какие платформы эффективнее
- Простые графики: средний охват по типу контента, тренд охватов за время
- Инсайт-карточки: «Кейсы дают x3 охватов по сравнению с мемами», «Reels работают лучше текста на 40%»

### Банк идей
- Быстрое добавление идеи для поста (текстовое поле, 1 тап)
- Список идей с возможностью отметить «использовано»
- Когда садишься создавать — не начинаешь с нуля, а выбираешь из своих заготовок
- Интеграция с рулеткой: если есть идеи в банке, рулетка может предложить одну из них

### Социальный слой
- **Share-карточка**: генерация PNG-карточки «Мой стрик {N} дней, уровень {название}, {XP} XP» для публикации в сторис
- Кнопка «Поделиться» на главном экране и в достижениях
- Open Graph мета-теги для красивого превью при шеринге ссылки на приложение

## Техническая реализация

### Инициализация проекта
- Vite + React + TypeScript
- Tailwind CSS (заменить все inline-стили из прототипа)
- Структура папок:
  ```
  src/
    components/
      Dashboard/       — главный экран (стрик, XP, недельный календарь, ритуал)
      LogPost/         — логирование нового поста
      Roulette/        — smart-рулетка с шаблонами
      Stats/           — статистика, достижения, тепловая карта
      Challenges/      — активные челленджи с прогрессом
      IdeaBank/        — банк идей
      Analytics/       — аналитика охватов
      ShareCard/       — генерация share-карточки
      Layout/          — навигация, обёртка
    hooks/
      useGameState     — основное состояние игры
      useStreak        — логика стриков и decay
      useAchievements  — проверка и анлок достижений
      useChallenges    — активные челленджи
      useDailyQuest    — генерация daily quest
      useSeason        — логика сезонов
    data/
      contentTypes.ts  — типы контента и XP
      levels.ts        — уровни прокачки
      achievements.ts  — все достижения с условиями
      rouletteTasks.ts — задания рулетки с шаблонами
      challenges.ts    — определения челленджей
      dailyQuests.ts   — пул daily quest
      phrases.ts       — мотивационные фразы
    utils/
      xp.ts            — калькулятор XP (бонусы, комбо, decay)
      dates.ts         — дата-хелперы
      shareCard.ts     — генерация PNG через canvas
      sounds.ts        — управление звуками
    types/
      index.ts         — все TypeScript интерфейсы
    App.tsx
    main.tsx
  ```

### Хранение данных
- localStorage с миграцией версий (ключ `ship_it_v2`, автоматический апгрейд с v1)
- Интерфейсы: GameState, LogEntry, ContentType, Level, Achievement, Challenge, Season, Idea

### UX
- **PWA**: манифест + service worker → установка на телефон
- **Push-уведомления**: вечернее напоминание (Notification API)
- **Анимации**: Framer Motion для переходов, анлоков, рулетки
- **Тактильный отклик**: navigator.vibrate() при логировании и анлоке
- **Свайп-навигация** между вкладками
- **Звуки**: короткие эффекты при логировании, анлоке, рулетке (отключаемые в настройках)

### Навигация (вкладки)
1. 🏠 Главная — дашборд, стрик, XP, ритуал, быстрые действия
2. ➕ Пост — логирование нового поста
3. 🎰 Рулетка — smart-рулетка с шаблонами
4. ⚔️ Челленджи — активные челленджи с прогрессом
5. 📊 Стата — аналитика, достижения, тепловая карта, банк идей
