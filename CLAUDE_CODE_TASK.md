# SHIP IT — Задача для Claude Code

## Цель
Превратить прототип геймифицированного трекера контента «Ship It» (React-компонент) в полноценное мобильное веб-приложение и задеплоить его.

## Исходник
Прототип: `ship-it.jsx` (приложен в проекте)
Это рабочий React-компонент со следующими механиками:
- **Стрик** — непрерывная серия дней с публикациями, бонусы за 3+ и 7+ дней
- **RPG-прокачка** — XP за разные типы контента, 7 уровней, 9 достижений
- **Рулетка форматов** — рандомное задание из 18 вариантов под нишу лидгена

## Что нужно сделать

### 1. Инициализация проекта
- Создать проект на Vite + React + TypeScript
- Настроить Tailwind CSS (заменить inline-стили)
- Структура папок:
  ```
  src/
    components/    — UI-компоненты (Dashboard, LogPost, Roulette, Stats, Layout)
    hooks/         — useGameState, useStreak, useAchievements
    data/          — contentTypes, levels, achievements, rouletteTasks (константы)
    utils/         — xp-калькулятор, дата-хелперы
    types/         — TypeScript типы
    App.tsx
    main.tsx
  ```

### 2. Перенос и рефакторинг
- Разбить монолитный компонент на отдельные модули по структуре выше
- Перевести все стили в Tailwind-классы
- Типизировать всё через TypeScript (интерфейсы для GameState, LogEntry, ContentType, Level, Achievement)
- Хранение данных: localStorage с миграцией версий (ключ `ship_it_v2`, автоматический апгрейд с v1)

### 3. Улучшения UX
- **PWA**: манифест + service worker → установка на телефон как приложение
- **Уведомления**: напоминание вечером если нет поста (через Notification API, с разрешением пользователя)
- **Анимации**: Framer Motion для переходов между вкладками, анлока достижений, рулетки
- **Тактильный отклик**: navigator.vibrate() при логировании поста и анлоке
- **Свайп-навигация** между вкладками (опционально, если несложно)
- **Тёмная тема** — уже есть, но добавить CSS-переменные для потенциальной смены темы

### 4. Дополнительные фичи (реализовать все)
- **Ежедневный квест**: каждый день генерируется 1 мини-задание (бонус +15 XP за выполнение)
- **Комбо-система**: 2 поста за день = x1.5 XP за второй, 3 = x2.0
- **Штраф за пропуск**: пропустил день = -10% от накопленного XP (мотивация не ломать стрик)
- **Экспорт статистики**: кнопка «Поделиться» → генерирует картинку-карточку со статами (canvas → PNG)
- **График активности**: тепловая карта по типу GitHub contributions за последние 90 дней
- **Звуки**: короткие звуковые эффекты при логировании и анлоке (отключаемые)

### 5. Деплой
- Платформа: **Vercel** (бесплатно, быстро, автодеплой)
- Домен: настроить кастомный домен если есть, иначе оставить `ship-it-game.vercel.app`
- Настроить:
  - `vercel.json` с правильными редиректами для SPA
  - Мета-теги для PWA (manifest.json, иконки 192x192 и 512x512)
  - Open Graph теги (для шеринга ссылки)

### 6. Качество
- Lighthouse score: целевой 90+ по всем категориям
- Адаптив: идеально на мобилке (375px), нормально на десктопе
- Нет ошибок в консоли
- Работает офлайн (service worker кеширует assets)

## Дизайн-система
- Фон: `#0a0a0f`
- Акцент: `#e2c044` (золото)
- Успех: `#4ade80`

- Шрифт: JetBrains Mono (основной), системный моноширинный как фоллбэк

- Стиль: тёмный, минималистичный, «терминал + игра»
